# Business Model Canvas — Phase 1: 솔로프레너 워크 OS

> **제품명**: Manage Agent (가칭)
> **Phase**: 1 — 솔로프레너 타겟 MVP
> **작성일**: 2026-02-05
> **다음 Phase**: Phase 2 (AI Life Coach, B2C 확대) → Phase 3 (AI Agent Orchestrator)

---

## 문제

1. **도구 파편화**: Notion + Todoist + Toggl + Excel 등 4-5개 앱을 오가며 컨텍스트 스위칭
2. **의사결정 피로**: "오늘 뭘 해야 하지?"를 매일 아침 스스로 결정해야 하는 인지 부담
3. **성과 체감 부재**: 일한 시간 대비 성과를 수치로 확인하기 어려움 → 번아웃

## 솔루션

1. Plan→Focus→Review 통합 루프를 하나의 앱에서 제공
2. AI가 오늘의 우선순위를 제안하고, 자연어로 태스크 생성
3. 포모도로 기반 집중 시간 추적 + 주간/월간 리포트로 성과 시각화

## 경쟁우위

- 대부분의 생산성 앱은 "계획" 또는 "실행" 중 하나만 잘함. **Plan→Focus→Review 3단계가 하나의 루프로 연결**된 앱은 거의 없음
- AI 자연어 태스크 파싱이 이미 작동 중 (MVP 검증 완료)
- 1인 개발자 구조 → 빠른 의사결정, 낮은 번레이트

## 가치 맵

### 제품 & 서비스

- 목표/태스크 관리 (칸반, 캘린더, 우선순위)
- 포모도로 집중 타이머 (태스크 연동)
- AI 자연어 태스크 생성
- 주간 리뷰 & 생산성 히트맵
- 루틴/습관 트래커
- (추가 예정) 클라이언트 관리, 인보이스, 수익 대시보드

### 혜택창출방안

- AI가 매일 아침 "오늘의 추천 일정" 제안 → 의사결정 부담 제거
- 주간 리포트로 성과 수치화 → "이번 주 42시간 집중, 달성률 78%"
- 하나의 앱에서 전체 워크플로우 처리 → 컨텍스트 스위칭 제거

### 불만해소방안

- 복잡한 설정 없이 바로 시작 가능 (온보딩 3분 이내)
- 기존 도구 데이터 임포트 지원 (Notion, Google Calendar)
- 무료 티어로 핵심 기능 충분히 사용 가능 → 결제 압박 없음

## 고객 세그먼트

### 고객활동

- 매일 아침 오늘 할 일을 정리하고 우선순위를 정함
- 여러 프로젝트/클라이언트를 동시에 관리
- 주말에 이번 주를 돌아보며 다음 주를 계획
- SNS/유튜브/뉴스레터로 자기 브랜딩 및 고객 획득

### 혜택 (고객이 원하는 것)

- 적은 시간으로 더 많은 성과
- 일과 삶의 경계 명확화
- 성장을 수치로 확인
- 번아웃 없는 지속 가능한 워크 리듬

### 불만 (고객의 현재 고통)

- 도구가 너무 많음 (구독료 월 $30-50)
- Notion은 유연하지만 세팅에 시간이 너무 듦
- Todoist/Things는 체크리스트일 뿐, "실행"을 도와주지 않음
- 시간 추적 앱은 귀찮아서 결국 안 쓰게 됨

### 타겟 세그먼트

| 순위 | 세그먼트 | 특징 |
|------|----------|------|
| 1차 | 프리랜서 개발자/디자이너 | 도구 친화적, 높은 지불의사 |
| 2차 | 1인 콘텐츠 크리에이터 | 유튜버, 뉴스레터 운영자 |
| 3차 | 코치/컨설턴트 | 클라이언트 관리 니즈 강함 |

공통: 혼자 일하며 월 $200만원+ 수익, 자기 시간을 직접 관리해야 하는 사람

## 핵심 파트너

- **Vercel / Neon**: 인프라 (서버리스 호스팅 + DB)
- **OpenAI**: AI 기능 엔진
- **솔로프레너 커뮤니티 리더 / 인플루언서**: 초기 배포 채널
- **Product Hunt / Indie Hackers**: 런칭 플랫폼
- **Stripe**: 결제 인프라

## 핵심 활동

- 제품 개발 및 반복 개선 (주간 릴리즈)
- 초기 사용자 인터뷰 & 피드백 수집 (10명의 파워유저)
- 콘텐츠 마케팅 (솔로프레너 생산성 관련 블로그/트위터)
- AI 기능 고도화 (자동 일정 배치, 패턴 기반 추천)

## 고유의 가치제안

> **"계획하고, 집중하고, 돌아보는 하루의 전체 사이클을 AI와 함께 하나의 앱에서."**
>
> 흩어진 5개 도구를 하나로 통합하고, AI가 매일의 의사결정 부담을 줄여주어
> 솔로프레너가 **실제로 중요한 일에 집중**할 수 있게 돕는다.

## 고객 관계

- **셀프서비스 + 커뮤니티** (Discord/Slack 채널)
- 주간 뉴스레터 (제품 업데이트 + 생산성 팁)
- 초기: 1:1 온보딩 콜 (첫 50명 유료 사용자)
- 인앱 피드백 위젯 → 기능 요청 투표

## 핵심 자원

- 이미 작동하는 MVP (배포 완료, 핵심 루프 검증됨)
- AI/Claude/Codex 활용 개발 역량 (빠른 개발 사이클)
- Next.js + Prisma + Vercel 풀스택 기술 스택
- (필요) 초기 사용자 커뮤니티, 디자인 역량
